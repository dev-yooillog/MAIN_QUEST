# 🚕 NYC Taxi Data Cleaning & Analysis

### 프로젝트 개요
* **목적:** 1,200만 건 이상의 대용량 뉴욕 택시 데이터를 정제하여 데이터 품질을 높이고, 유의미한 주행 패턴을 분석합니다.
* **주요 도구:** `Python`, `Pandas`, `Seaborn`, `Matplotlib`, `Google Colab`

---

### 데이터 정제 (Data Cleaning) 과정
데이터의 신뢰도를 확보하기 위해 다음과 같은 **무결성 검증** 및 전처리를 수행했습니다.

#### 1. 결측치 및 중복 처리
* `fare_amount` 누락 데이터 제거 및 중복 기록 삭제

#### 2. 파생 변수 생성
* 주행 시간(`duration`) 및 평균 속도(`speed`) 산출을 통한 물리적 타당성 검토

#### 3. 이상치(Outlier) 제거 기준
* **속도:** 시속 105km(65mph)를 초과하는 비현실적 데이터 삭제
* **인원:** 승객 수 0명 또는 7명 이상의 데이터 필터링
* **요금:** $0 이하 또는 $500 이상의 극단적 결측값/오류 제거
* **시간:** 하차 시간이 승차 시간보다 빠른 '시간 역행' 데이터 삭제

---

### 핵심 성과: 상관계수의 변화
데이터 정제 전후의 상관계수를 비교하여 전처리의 중요성을 입증했습니다.

| 분석 대상 (Distance - Fare) | 상관계수 (Correlation) | 비고 |
| :--- | :--- | :--- |
| **정제 전 (Raw Data)** | **0.001** | 관계 없음 (데이터 오염 상태) |
| **정제 후 (Cleaned Data)** | **0.932** | **매우 강력한 정비례 관계 확보** |

> **Insight:** 초기 데이터에서는 이상치로 인해 거리와 요금의 상관관계가 전무했습니다. 하지만, 정제 후 0.93이라는 높은 수치를 기록하며 분석 가능한 수준의 데이터 품질을 확보했습니다.
