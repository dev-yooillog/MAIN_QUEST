# Audi Car Price Analysis & Data Transformation

## 1. 데이터 확인 및 문제 정의
- 원본 데이터 로드: 구글 코랩 환경에서 audi.xls 데이터셋 로드 및 구조 파악
- 데이터 현황: 총 10,668개 행, 결측치 없음 확인
- 핵심 과제: 모델 학습을 위해 텍스트(Object) 데이터를 수치형으로 변환하고, 가격(price) 데이터의 통계적 편향성을 해결함

## 2. 범주형 데이터 변환 (Data Transformation)
- Label Encoding: model, transmission 컬럼을 고유 정수값으로 맵핑하여 데이터 차원 유지
- One-Hot Encoding: 서열이 없는 fuelType을 독립된 컬럼들로 분리하여 모델의 오해 방지
- 데이터 타입 최적화: 모든 범주형 변수를 int 타입으로 변환하여 연산 효율성 확보

## 3. 타겟 변수 분포 최적화 (Log Transformation)
- 비대칭성 식별: 시각화 결과, 낮은 가격대에 데이터가 쏠린 Right-skewed 분포 확인4
- 로그 변환 적용: np.log1p를 활용해 타겟 변수를 정규분포(종 모양) 형태로 보정
- 기대 효과: 이상치의 영향력을 줄여 모델의 학습 안정성 및 예측 정확도 향상

## 4. 데이터 시각화 및 검증 (EDA)
- 상관관계 분석: Heatmap을 통해 변수 간 상관계수 산출 및 시각화
- 주요 인사이트: * year(연식)와 가격 간의 강한 양의 상관관계(0.75) 확인
                  mileage(주행거리)와 가격 간의 뚜렷한 음의 상관관계(-0.68) 확인

- 데이터 무결성: 모든 피처가 수치형으로 변환되어 머신러닝 모델 투입이 가능한 상태임을 최종 확정

## 5. 결과 요약
- 데이터 정제: 10,668행의 무결성 유지 및 결측치 0건 처리
- 피처 엔지니어링: 3종의 범주형 인코딩 및 타겟 로그 변환 완료
- 최종 상태: 머신러닝 학습을 위한 최적화된 데이터셋
